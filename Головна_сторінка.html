html

HTML (HyperText Markup Language) — це мова розмітки, яка використовується для створення вмісту веб-сторінок. HTML використовується для опису структури та змісту сторінки, включаючи текст, зображення, відео, форми та інші елементи. Він визначає, як вміст має бути представлений та організований на сторінці.

css

CSS (Cascading Style Sheets) — це мова таблиць стилів, яка використовується для опису зовнішнього вигляду вмісту веб-сторінки: кольору, шрифту, розміру та інших аспектів відображення елементів сторінки. Він дозволяє розділяти структуру та зміст від зовнішнього вигляду сторінки, що спрощує її обслуговування та покращує її доступність.
